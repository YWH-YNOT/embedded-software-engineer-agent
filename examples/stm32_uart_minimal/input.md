# 输入示例：STM32 UART 最小 bring-up

## 项目背景

我在做一个基于 STM32F103C8T6 的小板，先只想把 UART1 日志口打通。

## 已提供信息

- 芯片：STM32F103C8T6
- 原理图：已提供
- 引脚表：已提供
- 官方资料：参考手册、数据手册、STM32Cube HAL
- UART1：PA9 为 TX，PA10 为 RX
- 当前目标：上电后通过 UART1 输出启动日志
- 当前策略：先用阻塞式发送，不做 DMA
- 波特率：115200
