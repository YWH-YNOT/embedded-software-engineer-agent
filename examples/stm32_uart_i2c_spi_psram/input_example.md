# 示例输入：STM32 + UART + I2C + SPI + PSRAM

本文件模拟用户第一次把资料和需求交给 Agent 时的输入。

## 1. 项目目标

我在做一个基于 STM32 的小型控制板，当前想让 AI 协助完成以下内容：

- UART1 作为调试日志口
- I2C1 驱动一个板载温度传感器
- SPI2 作为外部高速存储总线
- 外接 PSRAM 作为数据缓冲区

要求：

- 不要直接生成整包工程
- 先审查资料
- 如果关键技术信息缺失，必须先追问
- 按模块推进，先做单模块验证，再做系统集成

## 2. 已提供资料

| 资料 | 说明 |
| --- | --- |
| `RM0090` | STM32F4 参考手册 |

## 3. 当前已知硬件信息

| 项目 | 内容 |
| --- | --- |
| MCU | STM32F407VET6 |
| 板卡版本 | v1.2 |
| 主时钟 | 外部 8 MHz 晶振 |
| 调试接口 | SWD |
| UART1 | PA9/PA10，用于日志输出 |
| I2C1 | PB6/PB7，连接板载温度传感器 |
| SPI2 | PB13/PB14/PB15，外接 PSRAM |
| PSRAM 片选 | PB12 |
